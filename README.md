# SQL Data Warehouse Project

Developed a modern SQL-based Data Warehouse project implementing the Medallion Architecture using Bronze, Silver, and Gold layers for scalable and organized data processing. The project focuses on ETL pipeline development, data cleansing, transformation, and analytical modeling using SQL Server.

### Key Features

* Designed a multi-layered data warehouse architecture:

  * **Bronze Layer** for raw data ingestion
  * **Silver Layer** for data cleaning, validation, and transformation
  * **Gold Layer** for business-ready analytical tables and reporting
* Built ETL workflows using SQL stored procedures and transformation queries
* Implemented data quality checks, null handling, deduplication, and standardization techniques
* Performed data modeling and created optimized analytical datasets
* Used window functions, joins, aggregations, CTEs, and indexing concepts for efficient querying
* Structured the project to simulate real-world enterprise data engineering workflows

### Tech Stack

* SQL Server
* T-SQL
* Data Warehousing
* ETL Pipelines
* Medallion Architecture
